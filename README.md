## Technologies Used
- GitHub Pages for hosting- Jekyll for static site generation
- GitHub-flavored Markdown for content

## How to Update1. Edit the Markdown files in this repository
2. Commit and push changes
3. GitHub Pages automatically rebuilds the site (takes 1-2 minutes)

## Project Structure
- `_config.yml` - Site configuration- `index.md` - Homepage content
- `projects/` - Example Project details page
- `images/` - Images and static files
